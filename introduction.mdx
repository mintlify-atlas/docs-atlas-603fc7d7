Lebrijaleo is a modern web application developed as a Trabajo de Fin de Grado (TFG) project. It is built on Next.js 16 with the App Router, React 19, and Tailwind CSS v4, providing a solid foundation for a community-oriented web platform.

## Technology stack

Lebrijaleo is built on a modern, production-ready stack:

| Technology | Version | Purpose |
|---|---|---|
| [Next.js](https://nextjs.org) | 16.2.0 | React framework with App Router |
| [React](https://react.dev) | 19.2.4 | UI component library |
| [Tailwind CSS](https://tailwindcss.com) | v4 | Utility-first styling |
| [Geist](https://vercel.com/font) | latest | Typography (sans + mono) |

## Key features

<CardGroup cols={2}>
  <Card title="App Router" icon="route">
    Uses Next.js 16's App Router for file-system based routing with layouts, loading states, and error boundaries.
  </Card>
  <Card title="Dark mode" icon="moon">
    Built-in dark mode support using CSS custom properties that respond to the user's system preference.
  </Card>
  <Card title="Optimized images" icon="image">
    Uses `next/image` for automatic image optimization, lazy loading, and responsive sizing.
  </Card>
  <Card title="Modern fonts" icon="font">
    Geist Sans and Geist Mono loaded via `next/font` for zero layout shift and optimal performance.
  </Card>
</CardGroup>
